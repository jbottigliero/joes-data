# Examples with Chakra UI

Since this generator supports [MDX](https://mdxjs.com/) **and** uses [Chakra UI](https://v2.chakra-ui.com/), you can easily create interative experiences that match the rest of your portal them by importing components.


For example:

```js
import {Button} from '@chakra-ui/react';
<Button colorScheme='blue'>Button</Button>
```

A more advanced example, might use state...

function DrawerExample() {
  const { isOpen, onOpen, onClose } = useDisclosure()
  const btnRef = React.useRef()

  return (
    <>
      <Button ref={btnRef} colorScheme='teal' onClick={onOpen}>
        Open Drawer
      </Button>
      <Drawer
        isOpen={isOpen}
        placement='right'
        onClose={onClose}
        finalFocusRef={btnRef}
      >
        <DrawerOverlay />
        <DrawerContent>
          <DrawerCloseButton />
          <DrawerHeader>Hello, world.</DrawerHeader>

          <DrawerBody>
            <Input placeholder='Type here...' />
          </DrawerBody>

          <DrawerFooter>
            <Button variant='outline' mr={3} onClick={onClose}>
              Cancel
            </Button>
            <Button colorScheme='blue'>Save</Button>
          </DrawerFooter>
        </DrawerContent>
      </Drawer>
    </>
  )
}
