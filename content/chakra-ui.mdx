# Examples with Chakra UI

Since this generator supports [MDX](https://mdxjs.com/) **and** uses [Chakra UI](https://v2.chakra-ui.com/), you can easily create interative experiences that match the rest of your portal by importing components.

<br />

For example:
<br />

```js

import {Button} from '@chakra-ui/react';

<Button>Button</Button>

```

A more advanced example, might even use state...

export function DrawerExample() {
  const { isOpen, onOpen, onClose } = useDisclosure()
  const btnRef = React.useRef()

  return (
    <>
      <Button ref={btnRef} colorScheme='teal' onClick={onOpen} my={4}>
        Open Drawer
      </Button>
      <Drawer
        isOpen={isOpen}
        placement='right'
        onClose={onClose}
        finalFocusRef={btnRef}
      >
        <DrawerOverlay />
        <DrawerContent>
          <DrawerCloseButton />
          <DrawerHeader>Hello, world.</DrawerHeader>

          <DrawerBody>
            <Input placeholder='Type here...' />
          </DrawerBody>

          <DrawerFooter>
            <Button variant='outline' mr={3} onClick={onClose}>
              Cancel
            </Button>
            <Button>Save</Button>
          </DrawerFooter>
        </DrawerContent>
      </Drawer>
    </>
  )
}
